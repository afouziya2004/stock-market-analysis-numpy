# Stock Market Statistical Analysis using NumPy

A data science project that performs statistical and volatility
analysis on stock market data using NumPy.

## Dataset
NSE Stock Market daily price data (Open, High, Low, Close, Volume)

## Key Analysis
- Daily returns calculation
- Mean, median, variance, standard deviation
- Volatility analysis
- Vectorized numerical operations

## Tech Stack
- Python
- NumPy
- Matplotlib

## Project Structure
- data/ : stock dataset
- src/ : NumPy analysis modules
- notebooks/ : exploratory analysis
- visuals/ : generated plots

## How to Run
python src/statistical_metrics.py  
python src/volatility_analysis.py
